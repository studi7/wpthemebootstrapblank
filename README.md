# wpthemebootstrapblank
Blank Theme for Wordpress based on Bootstrap

## Features
- Custom post type projects with taxonomies
- Basic SEO
- Breadcrumb
- Use holder.js library if thumbnail not exist
- 404 template
- Boostraped menu
